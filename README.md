# वेग

Vega (वेग) is developer friendly project scaffolding tool to speed up development process.


## Installation

Via **Go Get**:

```
go get github.com/srijanone/vega
```
---

## Getting Started

```console
vega init
vega starterkits list
vega create --starterkit <starterkit>
vega up
vega down
```
---

## Development

`go run main.go`
